# AdamW 性能对比汇总报告

> 数据来源：`docs/ADAMW_C.md` + `docs/ADAMW_K.md` + `docs/ADAMW_D.md`
> 每种配置 3 个独立运行数据（三位小伙伴各测一次），取中位数
> 测试模型：MNIST MLP (784→512→Tanh→256→Tanh→10)
> 优化器：AdamW (lr=0.001, β₁=0.9, β₂=0.999, ε=1e-8, wd=0.01) · 4 epochs · batch_size=128

---



## 测试配置

| 配置项     | 值                                                   |
| ---------- | ---------------------------------------------------- |
| 模型       | MLP（784→512→256→10），Tanh 激活                     |
| 数据集     | MNIST                                                |
| 优化器     | AdamW（lr=0.001, β1=0.9, β2=0.999, ε=1e-8, wd=0.01） |
| 调度器     | ConstantLR（base_lr=0.001）                          |
| Batch Size | 128                                                  |
| Epochs     | 4                                                    |
| PyTorch    | 2.x + torch.compile(max-autotune)，编译开销已排除    |

## 最高准确率

| 模式 | TR4 中位数 | PyTorch 中位数 | 差距 |
|------|-----------|---------------|------|
| CPU  | **97.60%** | 97.35% | TR4 +0.25% |
| GPU  | **97.60%** | 97.40% | TR4 +0.20% |
| AMP  | **97.50%** | 97.29% | TR4 +0.21% |

---



## 训练用时

| 模式 | TR4 中位数 (s) | PyTorch 中位数 (s) | TR4 vs PyTorch |
|------|---------------|--------------------|----------------|
| CPU  | 11.16         | 19.04              | TR4 快 **1.71×** |
| GPU  | 1.86          | 10.70              | TR4 快 **5.75×** |
| AMP  | 1.75          | 14.19              | TR4 快 **8.11×** |

> PyTorch GPU/AMP 已排除 `torch.compile(max-autotune)` 编译预热时间，仅计 4 个 epoch 纯训练开销。
