# Tensor打印功能

**版本**: V3.6.13
**更新日期**: 2026-01-01
**作者**: 技术觉醒团队

## 概述

renAIssance框架的Tensor打印功能完全兼容PyTorch的打印格式，确保开发者在使用我们的框架时能够获得与PyTorch一致的视觉体验。

## 打印方法

### 方法1: `print()` 方法

```cpp
#include "renaissance.h"

using namespace tr;

auto& cpu = DeviceManager::instance().cpu();

// 创建张量
Tensor tensor = cpu.randn(Shape({2, 3, 4}), 0.0f, 1.0f, DType::FP32);

// 打印张量（默认4位小数）
tensor.print("tensor");

// 打印张量（自定义精度）
tensor.print("tensor", 6);  // 6位小数
```

### 方法2: `print_tensor()` 通过PythonSession

使用PyTorch的打印引擎来获得100%一致的输出格式：

```cpp
#include "renaissance/utils/python_session.h"

PythonSession session;
session.start();

Tensor tensor = cpu.randn(Shape({2, 3, 4}), 0.0f, 1.0f, DType::FP32);

// 获取PyTorch格式的打印文本
std::string pytorch_output = session.print_tensor(tensor);

// 输出到控制台
std::cout << pytorch_output << std::endl;

session.stop();
```

## 支持的张量维度

### 0D张量（标量）

```cpp
Tensor scalar = cpu.full(Shape(), 3.14159f);
scalar.print("scalar");
```

输出：
```
scalar: 3.1416
```

### 1D张量

```cpp
Tensor tensor1d = cpu.randn(Shape({10}), 0.0f, 1.0f, DType::FP32);
tensor1d.print("tensor1d");
```

输出（第8个元素后自动换行）：

```cpp
Tensor tensor2d = cpu.randn(Shape({3, 4}), 0.0f, 1.0f, DType::FP32);
```

输出（每行缩进8个空格）：

## 精度控制

`print()` 方法支持自定义小数位数：

```cpp
tensor.print("tensor", 2);  // 2位小数
tensor.print("tensor", 4);  // 4位小数（默认）
tensor.print("tensor", 6);  // 6位小数
```

## 数据类型支持

打印功能支持所有数据类型：

- **FP32**: 单精度浮点数，右对齐，正数前加空格
- **BF16**: BF16格式（转换为FP32显示）
- **INT32**: 32位整数，宽度7位右对齐
- **INT8**: 8位整数，宽度7位右对齐

## 实现细节

- **格式一致性**: 完全匹配PyTorch的打印格式，包括缩进、换行、逗号位置
- **数字对齐**: 浮点数右对齐，正数前加空格，确保列对齐
- **自动换行**: 1D张量在第8个元素后自动换行
- **内存友好**: 打印大张量时只显示摘要信息
