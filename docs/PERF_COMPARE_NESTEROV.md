# SGD+Nesterov 性能对比汇总报告

> 数据来源：`docs/NESTEROV_C.md` + `docs/NESTEROV_K.md` + `docs/NESTEROV_D.md`
> 每种配置 3 个独立运行数据（三位小伙伴各测一次），取中位数
> 测试模型：MNIST MLP (784→512→Tanh→256→Tanh→10)
> 优化器：SGD + Nesterov (lr=0.01, momentum=0.9, wd=0.01) · 4 epochs · batch_size=128

---

## 测试配置

| 配置项     | 值                                                   |
| ---------- | ---------------------------------------------------- |
| 模型       | MLP（784→512→256→10），Tanh 激活                     |
| 数据集     | MNIST                                                |
| 优化器     | SGD + Nesterov（lr=0.01, momentum=0.9, wd=0.01）     |
| 调度器     | ConstantLR（base_lr=0.01）                           |
| Batch Size | 128                                                  |
| Epochs     | 4                                                    |
| PyTorch    | 2.x + torch.compile(max-autotune)，编译开销已排除    |

---

## 最高准确率

| 模式 | TR4 中位数 | PyTorch 中位数 | 差距 |
|------|-----------|---------------|------|
| CPU  | **96.71%** | 95.10% | TR4 +1.61% |
| GPU  | **96.71%** | 95.10% | TR4 +1.61% |
| AMP  | **96.71%** | 95.10% | TR4 +1.61% |

---

## 训练用时

| 模式 | TR4 中位数 (s) | PyTorch 中位数 (s) | TR4 vs PyTorch |
|------|---------------|--------------------|----------------|
| CPU  | 10.66         | 19.10              | TR4 快 **1.79×** |
| GPU  | 1.81          | 10.47              | TR4 快 **5.78×** |
| AMP  | 1.66          | 14.02              | TR4 快 **8.45×** |

> PyTorch GPU/AMP 已排除 `torch.compile(max-autotune)` 编译预热时间，仅计 4 个 epoch 纯训练开销。
